✅ Section A: Fundamentals (1–40)
1. What is GitHub Actions primarily used for?

Answer: Automating CI/CD workflows such as build, test, deployment, and repository automation.

2. Where are workflow files stored?

Answer: .github/workflows/ directory.

3. Which file format does GitHub Actions use?

Answer: YAML (.yml or .yaml).

4. Can a repository have multiple workflows?

Answer: Yes.

Answer: on

Answer: workflow_dispatch

Answer: False.

Answer: runs-on

Answer: A server that executes GitHub Actions jobs.

Answer: A reusable automation component used in workflows.

Answer: False.

Answer: Executes shell commands.

Answer: Yes.
